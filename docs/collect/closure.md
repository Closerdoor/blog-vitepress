## 定义

- 闭包是函数与其定义时所处词法作用域的组合。
- 内部函数引用了外部函数变量，且该引用在外部函数执行结束后仍然有效，就是典型闭包场景。

## 作用

- 封装私有状态。
- 延长变量生命周期。
- 构造高阶函数与函数工厂。

## 风险

- 不必要的闭包会让对象长期被引用，增加内存占用。
- 闭包本身不是内存泄漏，问题在于“该释放的引用没有释放”。

## 最小示例

```js
function createCounter() {
  let count = 0

  return function () {
    count += 1
    return count
  }
}

const next = createCounter()
next() // 1
next() // 2
```

## 面试回答重点

- 先给定义，再讲应用场景。
- 说明闭包依赖词法作用域，而不是调用位置。
- 追问性能问题时，强调及时解除无用引用。
